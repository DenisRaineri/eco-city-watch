# Guia de Contribuição - EcoCity Watch

## Projeto Acadêmico UNIP

Este é um projeto acadêmico desenvolvido por Denis Raineri para a UNIP (8º Semestre).

## Estrutura de Desenvolvimento

### Padrões de Código

- Use TypeScript para todos os arquivos
- Siga as convenções do React Native e Expo
- Mantenha componentes pequenos e reutilizáveis
- Use nomes descritivos para variáveis e funções

### Estrutura de Commits

```
tipo(escopo): descrição

Exemplos:
feat(dashboard): adiciona métricas de qualidade do ar
fix(map): corrige problema de renderização de marcadores
docs(readme): atualiza instruções de instalação
```

### Testes

- Teste a aplicação em dispositivos iOS e Android
- Verifique compatibilidade web quando aplicável
- Teste funcionalidades de localização

### Melhorias Futuras

- Integração com APIs reais de dados ambientais
- Notificações push para alertas críticos
- Modo offline com cache de dados
- Análises mais avançadas com gráficos
- Gamificação para engajamento do usuário
